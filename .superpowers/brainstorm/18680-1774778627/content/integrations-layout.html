How agents know what's connected

When you connect Shopify, it gets stored in a NangoConnection record in MongoDB. Every time a conversation starts, the agent route loads those records and appends them to the system prompt — same pattern as today's native connections. The agent sees: CONNECTED PLATFORMS: Meta Ads, Google Ads, Shopify (mystore.myshopify.com), HubSpot — and new tools like get_shopify_orders become available automatically.
